## 📌 О проекте

Современный адаптивный сайт-портфолио продуктового дизайнера.

Проект реализован на основе готового дизайн-макета.  
Основной фокус — точная техническая реализация интерфейса,
адаптивная верстка и корректная production-сборка.

Сайт построен как полноэкранный одностраничный интерфейс
с секционной навигацией и popup-презентацией кейсов.

---

## 👤 Роль в проекте

**Frontend Developer — Solo Implementation**

- Верстка по готовому дизайн-макету (Figma)
- Адаптивная реализация интерфейса
- Реализация интерактивных элементов
- Настройка fullpage-навигации
- Popup-механика для кейсов
- Оптимизация изображений (WebP)
- Production-сборка и деплой
